# HackDaShit

A web-based tool for hacking arbitrary URLs and IP addresses.

## Tech Stack

- [React](https://react.dev/) with [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) for build tooling
- [Express](https://expressjs.com/) backend server
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [shadcn/ui](https://ui.shadcn.com/) components
- [Wouter](https://github.com/molefrog/wouter) for routing
- [Lucide React](https://lucide.dev/) for icons

## Development

Initially built with [Replit Agent](https://replit.com/products/agent), then continued with [Claude Code](https://www.claude.com/product/claude-code).

## Deployment

Deployed to [GitHub Pages](https://pages.github.com/) via [GitHub Actions](https://github.com/features/actions).
