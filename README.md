# HackDaShit

A web-based tool that can hack any URL or IP address.

## Tech Stack

- **React** with TypeScript
- **Vite** for build tooling
- **Express** backend server
- **Tailwind CSS** for styling
- **shadcn/ui** components
- **Wouter** for routing
- **Lucide React** for icons
